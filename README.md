# FineTuned-YOLOv8-for-rice-worm-detection
This project addresses rice worm infestation using YOLOv8 Nano for real-time detection. Fine-tuned on 818 augmented images with optimized hyperparameters, the model achieved mAP50 of 0.911 and mAP50-95 of 0.811, marking a significant improvement over the original YOLOv8
